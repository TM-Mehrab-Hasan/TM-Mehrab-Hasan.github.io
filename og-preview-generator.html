How to Generate og-preview.png:

  1. Open this file in Chrome or Edge browser
  2. Press F12 to open Developer Tools
  3. Click on the device toolbar icon (or press Ctrl+Shift+M)
  4. Set dimensions to exactly 1200 x 630
  5. Right-click on the preview area and select "Capture screenshot"
  6. Save the file as og-preview.png in the images folder

Alternatively, use a screenshot tool to capture the preview area at 1200x630 pixels.
